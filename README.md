# Study Planner

### 1. Пітч проєкту
[span_0](start_span)Study Planner — це інструмент для студентів, який вирішує проблему хаотичного планування завдань[span_0](end_span). [span_1](start_span)Продукт допомагає структурувати навчальний графік в одному місці та вчасно виконувати дедлайни[span_1](end_span).

### 2. Мета першої ітерації та ознаки успіху
[span_2](start_span)Мета — створити базову версію, де можна керувати завданнями[span_2](end_span).
**Ознаки успіху:**
* [span_3](start_span)Користувач може виконати головний сценарій без зайвих кроків[span_3](end_span).
* [span_4](start_span)Стабільна робота простої операції протягом тижня використання[span_4](end_span).

### 3. Ядро цінності (MVP)
[span_5](start_span)Мінімально життєздатний продукт включає один ключовий сценарій: можливість зафіксувати запис, побачити його в основному списку і позначити виконаним[span_5](end_span).

### 4. Карта стейкхолдерів
* **[span_6](start_span)Користувачі:** Студенти, які сприймають цінність продукту[span_6](end_span).
* **[span_7](start_span)Розробник (Я):** Той, хто впливає на реалізацію обмеженнями часу та ресурсів[span_7](end_span).
* **[span_8](start_span)Викладач:** Той, хто оцінюватиме досягнення цілей проєкту[span_8](end_span).
